# Mosoo API

The Mosoo Published Agent API lets approved callers create sessions on published Agents, send user events into those sessions, and replay the resulting event stream.

This documentation covers the public API surface for calling published Agents from external systems.

## How access works

Every request uses a Personal Access Token to identify the caller. The caller must still pass Organization membership and Agent access checks before they can use a published Agent.

Published Agents execute under the Agent owner's configured access boundary. The caller is recorded for admission, attribution, and session ownership, but the caller does not become the runtime execution owner.
