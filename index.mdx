# Groots API

The Groots Published Agent API lets approved callers create and drive sessions on published Agents.

This is the small public docs surface for M2D:

- [Quickstart](/quickstart): create a session, send a message, and replay events with curl.
- [Authentication and access](/auth-and-access): understand PAT caller identity, Agent access, and owner execution.
- API Reference: generated from the live OpenAPI document.

The API is PAT-authenticated and access-gated. A Personal Access Token identifies the caller account; Organization membership and Agent access decide whether that caller can use a specific Agent. Runtime execution still follows the Agent owner's configured access boundary.
